Help

Map your flight plan

Use the flight mapping tool to plan your flight, one leg at a time. After naming your flight and giving it a default speed, you can edit the speed of a particular leg by clicking on it's info.

Use the point mapping tool to mark a point of interest, such as a target, airfield or ground position. Hover over points to view their notes.

After placement, use the edit button to move and change elements on the map. Use the delete button to remove specific elements, and clear the entire map with the clear button .

Navigate the map

Use the jump to grid tool to jump to a specific, four digit grid number on the map.

Use the crop tool to fit the current flight plan in the window.

Share mission plans

Use the export tool to download a save file of the current mission plan.

Use the import tool to load a mission plan from a save file.

Use the stream tool to share missions in real time. Leaders can share their plans with viewers in a password protected session.

Configure the map

Access the settings to change the map and alter the style of the map text.
